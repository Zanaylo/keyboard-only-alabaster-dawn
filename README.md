# Keyboard Only — Alabaster Dawn

[Português](#português) · [English](#english)

---

## English

Comprehensive accessibility mod for **Alabaster Dawn** that lets you play the entire game **without a mouse**. Runs on top of the [Project Selene](https://github.com/project-selene) mod loader.

### Features

- **Lock-On**
- **Aim Stance (Tank Aim)**
- **Combat Art Turn Speed**
- **Mouseless Interact**
- **Keyboard Menu Navigation**
- **Keyboard-Only Aim Mode**

### Settings

All configurable under **Options → Accessibility → Keyboard Only**:

- Master switch (turns the entire mod on/off)
- Per-feature toggles
- **Aim Stance Turn Speed**
- **Aim-Hold Turn Speed**
- **Combat Art Turn Speed**
- Native **Lock-On** and **Aim Stance** keybinds under **Gameplay → Mouse & Keyboard → Keyboard Only**

### Install

1. Download `keyboard-only.mod.zip` from the latest release.
2. Drop it into the Alabaster Dawn `mods/` folder (create it if missing):
   ```
   <steam>/steamapps/common/Alabaster Dawn/mods/
   ```
3. Launch the game with Project Selene installed.
4. Select the mods folder and play.

## Português

Mod abrangente de acessibilidade para **Alabaster Dawn** que permite jogar o jogo inteiro **sem usar o mouse**. Roda em cima do mod loader [Project Selene](https://github.com/project-selene).

### Funcionalidades

- **Lock-On**
- **Aim Stance (Tank Aim)**
- **Combat Art Turn Speed**
- **Mouseless Interact**
- **Keyboard Menu Navigation**
- **Keyboard-Only Aim Mode**

### Configurações

Tudo configurável em **Options → Accessibility → Keyboard Only**:

- Master switch (liga/desliga o mod inteiro)
- Liga/desliga de cada feature individualmente
- **Aim Stance Turn Speed**
- **Aim-Hold Turn Speed**
- **Combat Art Turn Speed**
- Keybinds nativos do **Lock-On** e **Aim Stance** em **Gameplay → Mouse & Keyboard → Keyboard Only**

### Como instalar

1. Baixe o `keyboard-only.mod.zip` da última release.
2. Copie para a pasta `mods/` do Alabaster Dawn (cria a pasta se não existir):
   ```
   <steam>/steamapps/common/Alabaster Dawn/mods/
   ```
3. Inicie o jogo com o Project Selene instalado.
4. Selecione a pasta do Mod e inicie o jogo.
